Ging is quite simple to use, we must command him with proper clear instructions. And Ging takes care of the rest, if he does not understand the master's command, he show's it it the command prompt. The command prompt can be understood as the thought process of Ging. Ging understand some keywords and sequences of words. The Command promt will let us know if any error occurs. Remember that the assistant works on keywords, the user must use the same keywords so that Ging can understand.
